Sub-Dividable Behavior

An ability or restriction on the token where it can or cannot be subdivided from a single whole token into fractions, which are represented as decimal places.

Setting the base token Decimals property to 0 will make the token non-sub-dividable, any value greater will indicate how many fractions are possible where the smallest fraction is also the smallest ownable unit of the token.

Sub-dividable is common for crypto-currencies or tokens of fiat currency. For example, the US Dollar is sub-dividable to 2 decimal places, where a value like .42 is possible. Bitcoin, is sub-dividable up to 8 decimal places.

Non-subdividable is common for items where subdivision does not make sense, like a property title, inventory item or invoice.
