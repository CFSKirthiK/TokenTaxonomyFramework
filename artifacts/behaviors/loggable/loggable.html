Token Taxonomy Artifact

Loggable Behavior

A token class that implements this behavior will record log entries from its owner with a generic payload. These entries can be recorded stand alone and be given a unique identifier, EntryId, upon recording or these entries can be recorded in a series or group that will create a SeriesId and a EntryId, where all the entries will have a unique EntryId but have the same SeriesId.

Log entries can be queried by their EntryId or you can request an entire series with the SeriesId. The last recorded entry can also be requested without an Id and you can also request entries from a starting point to a finish point. For example, you could request entries 100 through 125, which will return the entries starting at position 100 through 125 or the last entry recorded up to 125.

To add entry query by any other property of the token, that property must be specifically defined and cannot be a property in the base token property list.

Behavior Invocations:

RecordEntryRequestA request to record an log entry.RecordEntryResponseA confirmation of recording the entry
GetEntryRequestA request to retrieve a specific Entry by its unique identifier.GetEntryResponseA response containing the specific log entry if found.
GetEntriesRequestA request to get a list of entries from log position A to log position B.GetEntriesResponseA response containing a list of entries from log position A containing entries to or up to the last position before log position B.
GetLastEntryRequestA request to retrieve the last log entry needing no parameters.GetLastEntryResponseA response container the last log entry if it exists.
CreateEntrySeriesRequestA request create a series of log entries.CreateEntrySeriesResponseA response containing a unique SeriesId that should be set for each entry's RecordEntryRequest message in the series.
GetEntrySeriesRequestA request retrieve all the log entries for a particular series by SeriesId.GetEntrySeriesResponseA response containing all the log entries for the requested SeriesId, if found.
