Token Taxonomy Artifact

SKU Property Set

A token class that implements this property set will have a SKU property or field with a Read/Query and Set control

Property Invocations:

GetSKURequestA request to read the SKU.GetSKURequestA response containing the string value of the SKU property.
SetSKURequestInput a new value for the SKU. It is recommended to use the Roles behavior with the SKU property to assign a role with the ability to invoke the Set control.SetSKUResponseA confirmation receipt or error may be returned to the invoker based on the outcome of the Set request.

Comments

Recommended that Roles be applied to this property.
